waitForElement(element)

returns a promise that allows to wait until a specified element is present on the DOM
Parameters:
  • jQuerySelector element A jquery selector that match an element not present in the DOM
Returns:
Ember.Test.promise
Example
waitForElement('.modal-dialog').then(function() {
    // code here will be executed when the element with the class "modal-dialog" is on the DOM
});
view source

waitForElementRemoval(element)

returns a promise that allows to wait until a specified element is not present anymore on the DOM
Parameters:
  • jQuerySelector element A jquery selector that match an element present in the DOM
Returns:
Ember.Test.promise
Example
waitForElementRemoval('.modal-backdrop').then(function() {
    // code here will be executed when the element with the class "modal-backdrop" is not on the DOM
});

waitMilliseconds(milliseconds)
