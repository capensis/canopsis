# Règles du projet

## Ligne de conduite - Code

Le code de l'interface de Canopsis respecte les règles de styles énoncées par AirBnB, disponibles [ici](https://github.com/airbnb/javascript).

L'utilisation d'une extension permettant l'intégration d'ESLint dans votre éditeur de texte/IDE est conseillée, afin de vous permettre d'identifier rapidement le code ne respectant ces règles.

Des commit-hooks ont été ajoutés, afin de vérifier le respect de ces règles. Si le code ajouté lors d'une tentative de commit ne les respecte pas, alors ce commit est bloqué. Des messages précisent les règles non respectées, ainsi que les zones de code qui posent problème.
