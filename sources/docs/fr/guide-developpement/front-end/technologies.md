# Technologies utilisées

- [VueJS](https://vuejs.org/): Framework Javascript
- [VueX](https://vuex.vuejs.org/): Gestionnaire d'état
- [Pug](https://pugjs.org/api/getting-started.html): Compilateur de template
- [WebPack](https://webpack.js.org)/[Babel](https://babeljs.io): Bundler/Compilateur
- [Vuetify](https://vuetifyjs.com/en/): Librairie de composants Material Design
- [Axios](https://github.com/axios/axios): Client HTTP
- [Lodash](https://lodash.com/): Librairie utilitaire
- [Moment](https://momentjs.com/): Librairie de gestion des dates
- [Vue-router](https://router.vuejs.org/): Gestion du routage
- [Vue-i18n](https://github.com/kazupon/vue-i18n): Gestion des traductions
