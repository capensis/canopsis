Menus
=====

Canopsis offers a set of built-in views that allows to show data and to configure the system. These built-in views are the more often used to set up the platform, before creating new custom views that shows specialized excerpts of data.

Menus are located on the top left of the interface, and allows to change or to manipulate the current view.
