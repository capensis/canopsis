# vues disponibles dans Canopsis

Les vues sont distribuées sous forme de briques, installables en fonction des besoins de chaque utilisateur. Certaines briques sont open-source, d'autres sont propriétaires et nécessitent une Licence de Canopsis.

## Briques open source

- [Calendrier](brick-calendar.md)
- [Bac à alarmes](brick-listalarms.md)
- [tableau de statistiques](brick-statstable.md)
- [Query Builder](brick-querybuilder.md)
